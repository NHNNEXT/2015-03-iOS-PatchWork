#2015-03-iOS-PatchWork
2015년 3학기 iOS 개경프 PatchWork팀 (김봉근, 김성한, 김기범)  

##서비스 개요  
간단한 조작만으로 순간의 감정을 표현할 수 있고,  
세계인들과 시각화된 감정을 실시간 공유 가능한 위치 기반 SNS
